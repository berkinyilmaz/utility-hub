# Utility Hub
One link to all 30 tools from the *30 Web Apps in 30 Days* series — search, filter and open any of them in a single click.

---

## Live Demo
https://utility-hub-navy-six.vercel.app/

---

## Features
- **30 tools in one place** — Day 1 → Day 30 of the series, side by side
- **Live search** — filter by name, blurb, category or day number
- **7 category filters** — Image · PDF · Text & Code · Generator · Time · Productivity · Web
- **Per-card actions** — open the live app (Vercel) or jump straight to the GitHub repo
- **Color-coded badges** — every category has its own subtle hue
- Apple-inspired dark UI with a soft hero gradient
- Sticky, glassy header with Instagram + GitHub shortcuts
- Fully responsive: 3 columns → 2 columns → 1 column
- Full keyboard accessibility, visible focus states
- Zero tracking, zero signup, zero ads

---

## Tech Stack
- React 19 (Vite)
- Pure CSS + CSS custom properties (Apple-inspired dark UI, no framework)
- Inline thin-line SVG icons (no icon library)
- Static deploy (Vercel)

---

## How It Works
1. Open the hub
2. Type a keyword or pick a category chip
3. Click **Open** on any card to launch the tool in a new tab
4. Click **Repo** to read the source on GitHub

> Everything runs locally in your browser. The hub is a static directory — clicks go directly to each tool's own Vercel deployment.

---

## Installation
```bash
git clone https://github.com/berkinyilmaz/utility-hub.git
cd utility-hub
npm install
npm run dev
```

---

## Privacy
Everything runs **locally in your browser**.
No data is stored, logged, or sent anywhere by the hub itself.
